Calendário

Calendários são elementos fundamentais e largamente utilizados em formulários para a entrada de dados por parte do usuário. Anteriormente, para suprir essa necessidade, foram desenvolvidos plugins utilizando JavaScript e CSS para criar um calendário, geralmente a partir de um input to tipo text. A jQuery UI fornece uma série de componentes como calendários, abas, botões e caixas de diálogo de fácil utilização.
